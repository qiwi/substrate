Basic types and interfaces. Typedoc: https://qiwi.github.io/substrate/types/

Install

 
yarn add @qiwi/substrate-types --dev
npm i @qiwi/substrate-types -D

Usage

 
import { IStringMap } from '@qiwi/substrate-types'

const foo: IStringMap = {
  bar: 'baz',
  qux: 'quux'
